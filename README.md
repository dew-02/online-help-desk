
# online-help-desk
Developed an Online Help Desk System,a web-based platform for submitting,managing, and resolving user support issues. Built with Java, JSP, MySQL, HTML, CSS, and JavaScript. Features include user and admin roles, ticket creation, status tracking, issue management, secure authentication, and a responsive interface for efficient help desk operations.
